# GAGYM Project Front-end/Back-end 소개

## 📍 프로젝트 소개

팀 프로젝트 명 : GAGYM

AWS 클라우드 기반
회원과 강사 간의 유기적 연계를 제공하는
피트니스 예약 서비스 시스템

## 📍 개발 인원 및 기간

개발 인원 : 프론트엔드/백엔드 3명

- FE: 박이슬 이효정 이희균
- BE: 박이슬 이효정 이희균

## 📍 기술스택

<!-- - Javascript (ES6+) -->

Front-end

- React with Hooks, React Router
- Typescript
- Next.js
- Redux-Saga

Back-end

- Spring Data JPA
- Spring Boot

Database

- PostgreSQL
- Redis
- RabbitMQ

AWS

- RDS
